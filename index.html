This is a front-end web project made with HTML,CSS,Bootstrap as a CSS framework and SASS as a preprocessor

Bootstrap is a free, open-source framework for front-end development that helps developers build websites and web apps more quickly and easily. It's designed to enable responsive development of mobile-first websites, and includes HTML and CSS based design templates for typography, forms, buttons, tables, navigation, modals, image carousels, and more. Bootstrap also saves developers time by eliminating the need to write CSS code.

Sass, which stands for Syntactically Awesome Style Sheets, is a scripting language used by front-end developers to create CSS style sheets. Sass is a CSS pre-processor that processes code and compiles it into CSS, which can then be used to style HTML elements. Sass is compatible with all versions of CSS and is an industry tool that makes front-end coding easier to maintain.

Syntactically Awesome Style Sheets (Sass) is a CSS preprocessing language that can improve CSS capabilities. It provides features like nesting, inheritance, and reusable styles, which can make code easier to write and maintain. Sass can also help organize stylesheets and reduce repetition. Sass includes features that don't exist in CSS, like variables, nested rules, mixins, imports, inheritance, and built-in functions. Sass can help developers write cleaner, more efficient code. It integrates seamlessly with vanilla CSS, keeping code organized and familiar. Sass is compatible with all versions of CSS. CSS frameworks are pre-prepared libraries of code that can help implement consistent website styling and layouts. Frameworks include pre-written, reusable code for common design elements and components that can be easily applied to base HTML.
